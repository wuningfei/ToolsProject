Docker
    Docker 是一个开源的应用容器引擎，开发者可以打包他们的应用以及依赖包到一个可移植的镜像中，
然后发布到任何流行的 Linux 或 Windows 机器上。
    1、容器技术的实质：通过各种手段，修改、约束一个"容器”进程的运行状态，按照用户的意图“误导”它能看到的资源，控制它的边界，
从而达到环境隔离，或者说虚拟化的目的。
    2、不要把 Docker 的概念与容器技术概念混淆，他们并不是等价的；
    3、容器的本质是一个进程。
    4、Docker命令
        查看本机images：docker images
        查看本机运行的container ： docker ps
        dockerhub上pull 镜像： docker pull 镜像名称
